Comandos Básicos do HTML5 & CSS3


Dominar os comandos básicos do HTML5 e CSS3 é essencial para qualquer desenvolvedor web.

Com essas linguagens, é possível criar desde páginas simples até aplicações web complexas, garantindo uma experiência agradável e consistente para os usuários em diferentes dispositivos. A combinação correta de estrutura (HTML5) e estilo (CSS3) é o primeiro passo para se tornar um desenvolvedor web competente e eficaz.


Abaixo veremos os comandos mais utilizados no HTML5 e no CSS3:

HTML5


Você pode usar quantas tags quiser para formatar o conteúdo. No entanto, existem algumas regras essenciais sobre tags HTML que você precisará seguir.


Uma tag HTML deve conter três partes:


  1. Uma tag de abertura - começando com um símbolo de abertura
  2. Conteúdo — instruções curtas para exibir o elemento na página;
  3. Uma tag de fechamento — terminando com um símbolo de fechamento

Algumas tags HTML podem ser abertas. Isso significa que elas não precisam ser fechadas com. Essas tags são normalmente usadas para metadados ou quebras de linha.

COMANDOS BÁSICOS


Elementos Estruturais
div Define uma divisão genérica ou seção no documento.
p Define um parágrafo de texto.
h1 até h6 Define cabeçalhos de diferentes níveis de importância.

Elementos de Conteúdo
a Cria um link para outra página ou recurso.
img Insere uma imagem na página.
ul e ui Criam listas não ordenadas.
ol Cria listas ordenadas.


CSS3


CSS (Cascading Style Sheets) é uma linguagem utilizada para estilizar elementos HTML. Com CSS, você pode definir como os elementos serão exibidos na página, incluindo cores, fontes, espaçamento e posicionamento.


Abaixo veremos alguns comandos básicos de CSS3 para começar.


Estrutura do documento CSS3



COMANDOS BÁSICOS


Seletores
.classe Permite aplicar estilos a elementos com uma determinada classe.
#id Permite aplicar estilos a um elemento com um ID específico.

Propriedades de Estilo
color Define a cor do texto.
background-color Define a cor de fundo de um elemento.
font-size Define o tamanho da fonte.
font-family Define a família de fontes a ser usada.
text-align Alinha o texto dentro de um elemento.

Modelo de Caixa
margin Define o espaço ao redor de um elemento.
padding Define o espaço interno entre a borda de um elemento e seu conteúdo.
border Define a borda de um elemento.

Estilos de Layout
display Define como um elemento deve ser exibido (block, inline, inline-block, etc.).
